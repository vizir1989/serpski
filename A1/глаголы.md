# 🇷🇸 ŠPARGALKA A1 — GLAGOLI (PREZENT)

Шпаргалка по глаголам сербского языка уровня **A1**  
Спряжение в настоящем времени (**prezent**)

---

## 🔹 I ГРУППА ГЛАГОЛОВ  
**Окончание в 1-м лице ед. числа: -IM**

### Схема
- ja **-im**
- ti **-iš**
- on/ona/ono **-i**
- mi **-imo**
- vi **-ite**
- oni/one/ona **-e**

### Примеры (A1)
| Инфинитив | ja |
|---|---|
| raditi | radim |
| učiti | učim |
| živeti | živim |
| govoriti | govorim |
| misliti | mislim |
| tražiti | tražim |
| voleti | volim |
| želeti | želim |
| videti | vidim |
| znati | znam |
| sećati se | sećam se |

---

## 🔹 II ГРУППА ГЛАГОЛОВ

| uzimati | uzimam |

---

## 🔹 III ГРУППА ГЛАГОЛОВ  
**Окончание в 1-м лице ед. числа: -EM**  
⚠️ Часто есть **изменение основы**

### Схема
- ja **-em**
- ti **-eš**
- on/ona/ono **-e**
- mi **-emo**
- vi **-ete**
- oni/one/ona **-u**

### Фонетические изменения (A1)
| Инфинитив | ja |
|---|---|
| pisati | pišem |
| kazati | kažem |
| lagati | lažem |
| plakati | plačem |
| skakati | skačem |
| kretati | krećem |
| stizati | stižem |

---

---

## 🔹 ОСОБЫЕ ГЛАГОЛЫ A1 (ОБЯЗАТЕЛЬНО ЗНАТЬ)

### 🔸 IĆI — идти / ехать / ходить
⚠️ В сербском **нет глагола «ехать»**

| Лицо | Форма |
|---|---|
| ja | idem |
| ti | ideš |
| on/ona/ono | ide |
| mi | idemo |
| vi | idete |
| oni/one/ona | idu |

---
## 🔸 JESTI — есть

| Лицо | Форма |
|------|-------|
| ja   | jedem |
| ti   | jedeš |
| on / ona / ono | jede |
| mi   | jedemo |
| vi   | jedete |
| oni / one / ona | jedu |

**Примеры:**
- Jedem kod kuće.
- Šta jedeš?

---

## 🔸 MOĆI — мочь, иметь возможность

| Лицо | Форма |
|------|-------|
| ja   | mogu |
| ti   | možeš |
| on / ona / ono | može |
| mi   | možemo |
| vi   | možete |
| oni / one / ona | mogu |

⚠️ **Важно**
- **može** — он может / можно / согласие  
- **možda** — возможно, может быть  

❌ *Možda li možeš?*  
✅ **Možeš li?**

---

## 🔸 HTETI — хотеть

| Лицо | Форма |
|------|-------|
| ja   | hoću |
| ti   | hoćeš |

| mi   | hoćemo |
| vi   | hoćete |

---

## 🔸 NE HTETI — не хотеть  
⚠️ **ОТДЕЛЬНЫЙ ГЛАГОЛ**

| Лицо | Форма |
|------|-------|
| ja   | neću |
| ti   | nećeš |
| on / ona / ono | neće |
| mi   | nećemo |
| vi   | nećete |
| oni / one / ona | neće |

❌ **ne hoću** — неправильно  
✅ **neću**

---

## 🔸 IĆI — идти / ехать / ходить

⚠️ В сербском **НЕТ отдельного глагола «ехать»**  
👉 **ići = идти + ехать**

| Лицо | Форма |
|------|-------|
| ja   | idem |
| ti   | ideš |
| on / ona / ono | ide |
| mi   | idemo |
| vi   | idete |
| oni / one / ona | idu |

**Примеры:**
- Gde ideš?
- Idemo na more.
- Idem u Beograd.

---

| on / ona / ono | je |

| oni / one / ona | su |

---

### Отрицание

### BITI в конструкциях **DA**

Используется после:
- morati  
- hteti  
- želeti  
- trebati  

| Лицо | Форма |
|------|-------|
| ja   | budem |
| ti   | budeš |
| on / ona / ono | bude |
| mi   | budemo |
| vi   | budete |
| oni / one / ona | budu |

**Примеры:**
- Moram da **budem** tamo.
- Hoću da **budemo** spremni.

---

## ✅ ИТОГ A1 ПО ГЛАГОЛАМ

Ты должен:
- различать **I / II / III группу**
- узнавать **III группу по форме ja**
- знать **особые глаголы наизусть**
- использовать **ići** для любого движения
- правильно говорить **hoću / neću**
