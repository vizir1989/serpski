# 🇷🇸 ŠPARGALKA A1 — SRPSKI JEZIK  
## Osnove (bez padeža)

---

## 1️⃣ PRISVOJNE ZAMENICE  
*(мой / твой / его / её / наш / ваш / их)*

📌 **Притяжательные местоимения согласуются с предметом**, а не с владельцем.

| Лицо | м.р. | ж.р. | ср.р. | мн.ч. |
|---|---|---|---|---|
| ja | moj | moja | moje | moji |
| ti | tvoj | tvoja | tvoje | tvoji |
| on | njegov | njegova | njegovo | njegovi |
| ona | njen | njena | njeno | njeni |
| mi | naš | naša | naše | naši |
| vi | vaš | vaša | vaše | vaši |
| oni | njihov | njihova | njihovo | njihovi |

### Примеры
- moj stan  
- moja kuća  
- njegovo dete  
- njihovi prijatelji  

---

## 2️⃣ GLAGOL **BITI** — JESAM / NISAM  
*(быть, настоящее время)*

### Утверждение
- ja **sam**
- ti **si**
- on / ona / ono **je**
- mi **smo**
- vi **ste**
- oni / one / ona **su**

### Отрицание
- ja **nisam**
- ti **nisi**
- on / ona / ono **nije**
- mi **nismo**
- vi **niste**
- oni / one / ona **nisu**

📌 `ne + biti` → **особая форма** (не *ne sam*)

---

## 3️⃣ PROŠLO VREME  
*(прошедшее время)*

### Формула
**jesam (sam) + l-форма**

| Род | Форма |
|---|---|
| м.р. | `-o` |
| ж.р. | `-la` |
| ср.р. | `-lo` |
| мн.ч. | `-li / -le` |

### Примеры
- ja sam radio  
- ona je došla  
- mi smo kupili  

---

### ❗ ВАЖНЫЕ ИСКЛЮЧЕНИЯ (A1)

| Инфинитив | Прошедшее время |
|---|---|
| ići | išao / išla |
| doći | došao / došla |
| otići | otišao / otišla |
| naći | našao / našla |
| reći | rekao / rekla |

---

## 4️⃣ NEGACIJA  
*(отрицание)*

### С глаголами
- ne radim  
- ne idem  
- ne razumem  

### С **biti**
- nisam  
- nisi  
- nije  
- nisu  

📌 `ne` **всегда перед глаголом**

---

## 5️⃣ SE-GLAGOLI  

`se` используется, когда:
- действие направлено на себя
- описывается состояние / самочувствие
- говорится об общем правиле или привычке

---

### 🔹 Возвратные (действие на себя) — A1

- učiti se — учиться  
- spremati se — готовиться  
- oblačiti se — одеваться  
- svlačiti se — раздеваться  
- tuširati se — принимать душ  
- kupati se — купаться  
- igrati se — играть  
- smejati se — смеяться  
- sećati se — помнить  

**Примеры:**
- Ja se spremam.
- Dete se igra.
- Smejem se.

---

### 🔹 Состояние / самочувствие — A1

- osećati se — чувствовать себя  
- bojati se — бояться  
- radovati se — радоваться  
- nervirati se — нервничать  
- opuštati se — расслабляться  

**Примеры:**
- Osećam se dobro.
- Bojim se.
- Radujem se vikendu.

---

### 🔹 Безличные / общие конструкции — A1

- kako se kaže? — как говорится?  
- ovde se radi — здесь работают  
- ovde se pije kafa — здесь пьют кофе  
- u Srbiji se jede puno mesa — в Сербии едят много мяса  
- tako se kaže — так говорят  

📌 `se` **не переводится буквально**

---

## 6️⃣ PRISVOJNI PRIDEVI  
*(притяжательные прилагательные от имён)*

| Тип имени | Суффикс | Пример |
|---|---|---|
| мужское | `-ov / -ev` | Milošev stan |
| женское | `-in` | Ljubičina torba |

📌 На A1 достаточно **узнавать**, не обязательно активно использовать

---

## 7️⃣ PRIDEVSKI SUFIKSI  
*(как в учебнике)*

| Основа | Суффикс | Пример |
|---|---|---|
| Srbija | -ski | srpski |
| Prag | -ški | praški |
| Amerika | -čki | američki |
| Nemačka | -čki | nemački |

📌 Значение: **относящийся к…**

---

## 🧠 A1-ЛОГИКА (КОРОТКО)

- глагол → лицо → окончание  
- прошедшее → **sam + l-форма**  
- отрицание → **ne перед глаголом**  
- притяжательные → **согласуются с предметом**

---

## ✅ ИТОГ
Это **полное грамматическое ядро A1 (часть 1)**:
- местоимения  
- глаголы (3 типа)  
- прошедшее время + исключения  
- отрицание  
- se-глаголы  
- суффиксы
